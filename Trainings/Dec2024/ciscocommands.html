Basic Switch Configuration Commands

  • enable

    • Goes from user exec mode to priv user exec mode

  • configure terminal

    • Opens terminal configuration

  • hostname {hostname}

    • Sets the devices name

  • enable secret {secret password}

    • Sets a password to access priv user exec mode

  • banner motd {message of the day}

    • Sets a default message of the day

  • Copy running-config startup-config | or for short | Copy run start

    • Saves the running configuration to the system startup configuration (Saves all changes made to the device configuration)
